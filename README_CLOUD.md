# KuroTask クラウドデプロイ手順書

PCを閉じてもスマホで常時使えるようにするためのデプロイ手順です。

## 1. Git のインストール
現在、あなたのPCに `git` がインストールされていないため、まずは以下のボタンからインストールをお願いします。
- [Git for Windows をダウンロード](https://git-scm.com/download/win)
※ 全て「Next」で進めて大丈夫です。

インストールが終わったら、このチャットで「インストールしたよ」と教えてください。その後のアップロード作業を私がサポートします。

## 2. GitHub へのアップロード（Gitインストール後）
1.  GitHub で新しくリポジトリ（名前例: `KuroTask`）を作成します。
2.  「Initialize this repository with a README」はチェック**しない**でください。
3.  作成後に表示される `https://github.com/あなたのUID/KuroTask.git` というURLをメモして、私に教えてください。
